Latest Project

Weather App

A responsive weather application built using HTML, CSS, and JavaScript. The app fetches real-time weather data from an API, allowing users to search for their city and view the current weather, temperature, humidity, and wind speed. It features a sleek and user-friendly design optimized for various devices.

QR Code Generator

A Python-based application that generates QR codes for text, URLs, or any custom input. Utilizing the qrcode library, this project creates high-quality QR codes that can be easily scanned with smartphones. It includes a simple interface and allows users to save the generated QR codes as image files for future use.

Rock-Paper-Scissors Game

A fun and interactive Rock-Paper-Scissors game built using HTML, CSS, and JavaScript. This project allows users to play against the computer, with random choices generated for the computer. The game keeps track of scores and features a simple, responsive interface for a smooth gaming experience.

Music Player

A feature-rich music player built using HTML, CSS, and JavaScript. This project allows users to play, pause, and navigate through songs with an interactive and visually appealing interface. The player includes functionalities like volume control, progress tracking, and playlist management, all designed to enhance the listening experience.

CRUD Operations in ASP.NET Web Forms

This project demonstrates how to perform CRUD (Create, Read, Update, Delete) operations using ASP.NET Web Forms in conjunction with SQL Server. Users can add new records, retrieve data, modify existing entries, and delete unwanted records, all with seamless integration between the front-end and the back-end.

University Management System using Java

This project involves the development of a comprehensive University Management System using Java. It handles various operations such as student registration, course enrollment, grade management, and scheduling. The system provides an intuitive interface for both students and administrators to manage academic processes efficiently.
